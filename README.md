# ColonyLink

A NeoForge mod for Minecraft 1.21.1 that bridges MineColonies and Applied Energistics 2.

## Features
- Resource GUI linked to your Builder's Hut via AE2 Wireless Access Point
- AE2 autocrafting integration
- Domum Ornamentum virtual crafting support
- Colony Link Redirector block with 120-slot buffer

## Dependencies
- MineColonies 1.1.1299+
- Applied Energistics 2 19.2.17+
- Domum Ornamentum 1.0.230+
- NeoForge 21.1.228
- Minecraft 1.21.1
