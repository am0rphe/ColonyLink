# ColonyLink

A NeoForge mod for Minecraft 1.21.1 that bridges MineColonies and Applied Energistics 2.

## Features

- Clipboard item linked to your Builder's Hut via AE2 Wireless Access Point
- AE2 autocrafting integration for missing builder resources
- Domum Ornamentum pattern encoding and virtual crafting support
- Warehouse Link Terminal — bidirectional bridge between MineColonies Warehouse and ME network
- Colony Link Redirector block with 27-slot Domum Pattern buffer

## Dependencies

- NeoForge 21.1.228+
- MineColonies 1.1.1299+
- Applied Energistics 2 19.2.17+
- Domum Ornamentum 1.0.233+

## Translations
